# webrtc-app-ts

Целью проекта является создание приложения, которое может позволить двум пользователям захватывать и передавать изображение и звук вкладки обозревателя (например Яндекс браузера) посредством WebRTC.

### Установка пакетов

```bash
yarn install
```

### Интерактивное обновление пакетов

```bash
yarn upgrade-interactive --latest
```

### Запуск сервера разработки

```bash
yarn dev
```
